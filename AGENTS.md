## Commands

- Check the `Makefile` for all commands and use `make <command>` to run commands when needed.

## Feedback

- When you need to validate Rust changes, run `make cargo-check`.

## Theme Colors

- DO NOT USE DEFAULT Tailwind CSS colors. Use the shadcn-ui color palette for consistency across the project.

## Ui Icons

- Use `hugeicons` instead of `lucide-icons` for UI icons.
